运行方法：
    若没环境需先安装运行环境：
        1. Python==3.7.6  # 重点，此基于Anaconda安装环境
        2. pip install -r environment.txt  # 安装包

    运行前配置：
        0. 显示器分辨率必须是1920*1080，其他分辨率可能会运行效果异常
        1. 打开并按需修改配置文件：config.py
        2. TLIAS播放器需事先选择记住密码登录。播放视频时，选择单个播放。
        3. 点击 双击运行的企鹅图标 开始运行
        git tag V1.2 -m '简化启动方式'
